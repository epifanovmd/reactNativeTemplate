# React Native Template

##### Stack:
  - Typescript
  - React
  - React Navigation
  - React-Redux
  - Redux Thunk
  - Redux Toolkit
  - i18next
  - Styled Components

### Installation
```sh
$ npm install -g @react-native-community/cli
$ npx react-native init ProjectName --template https://github.com/epifanovmd/reactNativeTemplate
$ cd ProjectName
```

### Run for Adnroid
```sh
$ yarn android
```

### Run for IOS
```sh
$ yarn ios
```
